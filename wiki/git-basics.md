# Git 基礎概念

Git 是版本控制工具，用來追蹤與管理程式碼變更。

## 主要概念

| 概念 | 說明 |
|------|------|
| commit | 儲存一個版本快照 |
| branch | 分支，可平行開發 |
| push | 上傳到遠端（如 GitHub） |
| pull | 從遠端下載更新 |

## 常用指令

```bash
git status          # 看目前狀態
git add .           # 加入所有變更
git commit -m "訊息" # 提交
git push            # 上傳
git pull            # 下載更新
```
