主題：學習 Git 基礎

Git 是一個版本控制工具。主要概念：
- commit = 儲存一個版本
- branch = 分支，可以平行開發
- push = 上傳到 GitHub
- pull = 從 GitHub 下載

常用指令：
git status - 看目前狀態
git add . - 加入所有變更
git commit -m "訊息" - 提交
git push - 推送到遠端
